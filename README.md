divi

Divi Destreet Theme

## Немного о работе форм feedback

Формы вставляются в страницу ввиде шоркодов
```[feedback tab="VAR"]```
где VAR может быть
* book - Выводит форму буклетов
* ad - Выводит форму заказа рекламы
* site - Заказ сайта
* cal - Заказ календаря.
